POST /api/orders — place an order, Saga advances it
$ curl -s -X POST http://localhost:8080/api/orders \
   -H "Content-Type: application/json" \
   -d '{ "customerId":"99999999-9999-9999-9999-999999999999",
         "items":[ {"productId":"22222222-2222-2222-2222-222222222222","quantity":2,"unitPrice":29.99} ] }'
{
  "id": "7f3a1c2e-9b4d-4e6a-8c1f-2a5b7d9e0f11",
  "customerId": "99999999-9999-9999-9999-999999999999",
  "status": "PLACED",
  "totalAmount": 59.98,
  "currency": "USD",
  "createdAt": "2026-06-19T06:45:12"
}

# a few seconds later — the Saga has carried it to DELIVERED:
$ curl -s http://localhost:8080/api/orders/7f3a1c2e... | jq '{status,totalAmount}'
{ "status": "DELIVERED", "totalAmount": 59.98 }
